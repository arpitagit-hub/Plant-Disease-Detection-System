# 🌿 Plant Disease Detection System

An AI-powered web application that detects plant diseases from leaf images using Machine Learning and Deep Learning techniques.

---

## 🚀 Features

* 📸 Upload plant leaf images
* 🤖 Predict plant disease using trained ML/DL model
* ⚡ Fast and user-friendly interface
* 🌐 Web-based application using Flask

---

## 🛠️ Tech Stack

* Python 🐍
* Flask 🌐
* HTML, CSS, JavaScript 🎨
* Machine Learning / Deep Learning 🤖

## ▶️ How to Run the Project

1. Clone the repository:

```
git clone https://github.com/your-username/plant_detection.git
```

2. Navigate to the project folder:

```
cd plant_detection
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the application:

```
python app.py
```

5. Open in browser:

```
http://127.0.0.1:5000/
```
